# Format officiel de l'épreuve CE1D Langues modernes (néerlandais)

Synthèse établie à partir des épreuves officielles 2022-2026 (enseignement.be),
téléchargées dans `docs/CE1D/` (ignoré par git : volumineux et non redistribuable —
repo public). Sert de calibrage au module examens blancs (M6, PRD §11/§16).

## Structure et barème (stable 2022 → 2026)

Total **/100**, seuil de réussite **50 %**. Deux journées : épreuve écrite
(Livret 1 + Livret 2) et épreuve orale.

| Partie                                     | Points | Contenu                                                                                                                                                                                                                                                        |
| ------------------------------------------ | ------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| CA — compréhension à l'audition (Livret 1) | /30    | 2 documents audio (dialogue, podcast…). Consigne : « Note, en français, N informations » (7 à 12 selon l'année). Chaque document est diffusé 2 à 3 fois (pistes séparées, dont versions courte/longue en 2026). Brut converti : CA1 et CA2 valent chacune /15. |
| CL — compréhension à la lecture (Livret 1) | /20    | 2 documents écrits NL (brochures, témoignages, article). Réponses ouvertes en français (« note N avantages », tableau lieux/activités…). CL1 = CL2 = /10.                                                                                                      |
| EE — expression écrite (Livret 2)          | /20    | **2 tâches de minimum 60 mots chacune, 50 minutes au total**, brouillon autorisé, grille de relecture fournie. EE1 = EE2 = /10 (grille /20 par tâche, ramenée).                                                                                                |
| EO — expression orale (dossier séparé)     | /30    | Fiche à contexte (8 fiches par session). **Tâche 1 sans interaction** : monologue, « donne au moins 5 informations différentes ». **Tâche 2 en interaction** : répondre aux questions de l'enseignant + **poser au moins 2 questions**. Support illustré.      |

## Grille officielle EE (par tâche, /20 → /10)

Deux grilles : la n°1 (normale) et la n°2 (plafond 8/20) si un « incontournable »
manque. Incontournables : **≥ 55 mots**, sujet respecté, ≥ 1 élément traité,
production compréhensible.

| Critère                                                                 | Points  |
| ----------------------------------------------------------------------- | ------- |
| Réalisation de la tâche (tous les éléments traités = 6, une partie = 3) | /6      |
| Type de production respecté                                             | /1      |
| Cohérence (organisation logique/chronologique)                          | /1      |
| Cohésion (pronoms, connecteurs simples : mais, ou, et, donc, car)       | /1      |
| Ponctuation simple (majuscule + point)                                  | /1      |
| Ressources lexicales A2- (4-3 / 2-1 / 0)                                | /4      |
| Orthographe du répertoire mémorisé                                      | /1      |
| Ressources grammaticales A2- (5-4 / 3-2 / 1-0)                          | /5      |
| **Total par tâche**                                                     | **/20** |

## Consignes types (utile pour générer des blancs au bon format)

- CA/CL : réponses **en français** (on teste la compréhension, pas la production) ;
  « informations précises et complètes » ; au-delà de N réponses, seules les N
  premières comptent.
- EE : contexte de communication réaliste (forum, message à un ami…), 2-3 éléments
  imposés, illustrations d'aide facultatives, compteur « Nombre de mots que tu as
  écrits » à remplir par l'élève.
- EO : champs thématiques des Socles (caractérisation personnelle, vie quotidienne,
  loisirs, voyages, santé, achats, nourriture, météo…) ; salutation attendue en
  début de tâche.

## Conséquences pour M6 (examens blancs)

- Le contenu officiel ne peut pas être embarqué dans l'app (repo public, droits
  FWB) : les blancs de l'app sont des épreuves **originales générées au même
  format et au même barème** ; les PDF officiels servent aux répétitions papier
  en conditions réelles (l'audio officiel peut être joué depuis `docs/CE1D/`).
- Chronométrage : EE = 50 min (officiel). CA/CL : le guide de passation n'est pas
  dans les téléchargements — approximations retenues : CA ≈ 25 min (rythmé par
  les écoutes), CL ≈ 40 min, ajustables.
- Correction : CA/CL par auto-correction contre le corrigé (comparer ses réponses,
  cocher juste/faux) ; EE par la grille officielle simplifiée en checklist ;
  EO enregistré au micro puis auto-évalué (+ écoute parent). Pas de juge
  automatique (PRD §13).
- Les guides de correction officiels ne sont pas inclus dans les zips
  téléchargés — à récupérer sur enseignement.be pour corriger les répétitions
  papier.
